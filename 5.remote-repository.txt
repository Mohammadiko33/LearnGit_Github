1. Repository چی بود ؟ 
    معنیش میشه مخزن
    در واقع همون محلی هست که سورس کد ما تغییرات و تاریخچش داره توش ذخیره میشه

------------------------------------------------------------------------------------
2. local Repository چیه ؟
    مخزنی که من توی سیستم خدم دارم و کسی دیگه ای بجز من بهش دسترسی نداره

------------------------------------------------------------------------------------
3. Remote Repository چیه ؟
    ریپازیتوری از راه دور ریپازیتوری هست که من میتونم به اشتراک بزارم به بقیه غیر از خدم هر کسی از هر جایی زمین میتونه به اون ریپازیتوری دسترسی داشته باشه

------------------------------------------------------------------------------------
4. این وصل کردن ریپازیتوری لوکال به ریموت دستور وارد میکنیم
        git Remote
          سپس این ادرس رو وارد میکنیم
                git remote add origin https://github.com/[user name]/[Repository name].git

------------------------------------------------------------------------------------
5. اینجا میگیم اخرین تغییراتی که بقیه به وجود اوردن رو از سرور بگیر و ارسال بکن توی لوکال من
    git pull origin master
------------------------------------------------------------------------------------
6. git push -u origin master 
    بار اور استفاده از این کامند برای ارسال فایل استفاده میکنیم و دفعه های بعدی 
        فقط کافیه 
        git push رو بزنیم
------------------------------------------------------------------------------------
7. وقتی که ما یک تغییراتی رو روی اوکال توی سیستم خودمون روی پروژه به وجود اوردیمبعد میایم یک کامیت ثبت بکنیم حالا میخایم تغییرات جدید رو بفرستیم روی ریپازیتوری گیتهاب برای اینکار از دستور زیر استفاده بکنیم
    git push
    پوش به معنای هل دادن هست 
------------------------------------------------------------------------------------
8. اما 
    pull 
    به معنای واکشی کردن هست گرفتن هست 
    اما چه موقه نیاز داریم از انی دستور استفاده بکنیم ؟ 
        مثلا کار تیمی که هر نفر روی یک بخش پروژه فعاله
        حالا وقتی که من میخام بیام و دوباره روی پروژه کار بکنم باید با استفاده از دستور گیت پول اخرین تغییرات پروژه از گیتهاب بگیرم
